# 📝 Exercício Prático - Aula 02

## 🎯 Objetivo
Aplicar conceitos de Kotlin: classes, funções, controle de fluxo

## 📋 Exercício
Criar classe `aula02.Produto` com:
- Atributos: nome (String), preco (Double), estoque (Int)
- Métodos: vender(quantidade), repor(quantidade), exibirInfo()
- Usar data class se apropriado

### Requisitos
- [ ] Classe aula02.Produto completa
- [ ] Método vender() reduz estoque (verificar se tem estoque)
- [ ] Método repor() aumenta estoque
- [ ] exibirInfo() mostra todos os dados
- [ ] Criar lista de 5 produtos diferentes
- [ ] Testar todas as operações

## 📤 Entregar
Arquivo `aula02.Produto.kt` em `exercicios/aula02/`
